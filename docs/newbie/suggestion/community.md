# 社区建议

加入神力科莎社区可以获得更多资源和帮助，以下是一些社区方面的建议。

## 社区平台

- **论坛**：访问官方论坛和国内相关论坛
- **社交媒体**：关注相关的社交媒体账号
- **Discord**：加入神力科莎相关的Discord服务器
- **直播平台**：观看其他玩家的直播，学习技巧

## 参与活动

- **线上比赛**：参加线上比赛，提高技术水平
- **社区活动**：参与社区组织的活动
- **MOD分享**：分享自己的MOD，获取反馈

## 资源获取

- **MOD网站**：了解常用的MOD下载网站
- **教程资源**：寻找适合新手的教程资源
- **工具软件**：使用社区开发的工具软件

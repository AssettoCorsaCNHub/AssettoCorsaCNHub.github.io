## 🌟 项目起源

《神力科莎》(Assetto Corsa / ACC) 是一款伟大的模拟赛车游戏，但对于国内新手玩家来说，存在着较高的门槛：
* **资源分散**：核心 Mod (如首都高 SRP) 往往分布在 Discord 或外网，下载极慢。
* **设置复杂**：Content Manager (CM)、CSP、Pure 等插件的配置让新手望而生畏。
* **语言障碍**：缺乏系统性的中文参数设置指南和常见问题解答 (FAQ)。

**AC-Hub China** 旨在建立一个开源、公益、持续更新的导航站，为中国玩家提供一站式的 Mod 下载引导、参数设置教程及避坑指南。

---

## 🛠️ 技术选型

本站基于 **[VitePress](https://vitepress.dev/)** 构建，具有以下优势：
* 🚀 **极速体验**：基于 Vite + Vue 3，页面加载如赛车般丝滑。
* 📝 **Markdown 驱动**：像写文档一样简单，支持嵌入 Vue 组件实现 FOV 计算器等小工具。
* 📦 **Git 部署**：完全托管于 GitHub / Vercel，通过 Git 进行版本控制，杜绝资料丢失。

---

## 📂 核心板块

* **[新手上路]**：从游戏购买到 CM 汉化的一键流程。
* **[Mod 专区]**：首都高 (SRP) 极速下载、车辆包、音效增强补丁。
* **[参数优化]**：针对不同配置电脑的 CSP 设置建议，以及 ACC 画面优化。
* **[常见问题]**：搜集并解决 99% 玩家会遇到的崩溃、报错问题。

## 🤝 参与贡献 (Join Us!)

这是一个开源社区项目，我们需要你的力量！

### 如何贡献？
1.  **文案补充**：发现有新的高质量 Mod 或更优的设置参数？直接提交 PR 修改 Markdown。
2.  **代码优化**：如果你精通 Vue 3，可以帮助优化页面 UI 或开发实用的赛车小工具。
3.  **纠错**：发现链接失效或描述错误？请提交 Issue。

### 提交步骤
1. Fork 本仓库。
2. 创建你的特性分支 (`git checkout -b feature/AmazingFeature`)。
3. 提交你的更改 (`git commit -m 'Add some AmazingFeature'`)。
4. 推送到分支 (`git push origin feature/AmazingFeature`)。
5. 开启一个 Pull Request。
